# Tia Toolbox

## TIA Toolbox is a set of Tools usable with TIA Portal.

You need to obtain a Licence to use TIA Portal Toolbox. The Licenese is personalized, for a Company Licence or Licence to use the Library in automation, please contanct us. 

### 1.) TIA Project Browser

Is a Dotnet Programm, usable in Windows, MacOs and Linux to view TIA Portal Projects. You could View Projects from Versions 10.5 to 20. 
Versions 10.5-13 have a little bit of limmeted suport.
You can use it to:
  - Get XML of Program Blocks (DB, FC, FB, ...). On Windows you can open them directly via the SIMATIC Automation Compare Tool
  - View WinCC/WinCC Unified Screens as HTML
  - View Images
  - View Variable Tables and Watch lists
  - Connect to a PLC and View the WatchTables and WatchLists online
